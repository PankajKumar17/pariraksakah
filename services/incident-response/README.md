# Incident Response Service

SOAR Orchestration Engine for CyberShield-X — automated incident response and remediation.

## Features
- Adaptive SOAR playbook execution engine (YAML-based)
- Automated playbooks: ransomware, credential theft, lateral movement, data exfiltration
- AI-powered investigation assistant (Claude API)
- Digital forensics automation with chain-of-custody
- Court-admissible evidence packaging

## Run locally
```bash
go run ./cmd/main.go
```
